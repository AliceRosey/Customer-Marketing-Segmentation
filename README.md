# Customer Marketing Segmentation

## Overview
This project applies unsupervised machine learning techniques to segment customers into meaningful groups based on behavioral and demographic characteristics. The analysis helps identify customer patterns that can support targeted marketing and business decision-making.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SciPy

## Machine Learning Techniques
- KMeans Clustering
- Hierarchical Clustering
- Principal Component Analysis (PCA)
- Silhouette Score Evaluation
- Dendrogram Visualization

## Files
- `Customer_Marketing_Segmentation.ipynb` – Main notebook containing data preprocessing, clustering, visualization, and analysis.
- `Clustering_Marketing.csv` – Dataset used for customer segmentation.

## Project Objectives
- Analyze customer behavior patterns.
- Perform customer segmentation using clustering techniques.
- Compare clustering methods using silhouette scores.
- Visualize clusters using PCA.
- Generate business insights from customer groups.

## Results
- Multiple clustering methods were evaluated and compared.
- Complete Linkage and Average Linkage achieved the highest silhouette scores.
- PCA visualization was used to understand cluster separation.
- The resulting customer segments provide insights for targeted marketing strategies and customer behavior analysis.

## Conclusion
The project demonstrates how unsupervised machine learning can be used to discover hidden patterns in customer data and create meaningful customer segments for marketing and business applications.
